# Hospital Management System Backend

Spring Boot backend for a Hospital Management System.

## Features

- User Authentication
- Registration API
- Login API
- Contact Enquiry API
- MySQL Integration
- REST APIs
- Role-Based Authentication

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## API Endpoints

### Authentication

POST /api/auth/register

POST /api/auth/login

### Contact

POST /api/contact

## Run Project

```bash
mvn spring-boot:run
